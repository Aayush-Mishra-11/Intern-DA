 # --- Customer Sales Analysis Report ---
 
  Data Exploration:
  - Loaded both sales_data.csv and customer_churn.csv
  - Checked for missing values and data types

  Data Cleaning:
  - Converted date column to datetime format
  - Extracted year, month, day, and day of week
  - Cleaned text data in Product and Region columns

  Customer Analysis:
  - Identified top 10 customers by lifetime value
  - Calculated average order value per customer
  - Analyzed regional distribution

  Sales Pattern Analysis:
  - Monthly sales aggregation (sum)
  - Product performance analysis (mean)
  - Customer order frequency (count)
  - This covers the required 3 aggregation types

  Data Merging:
  - Merged sales data with customer churn data using inner join
  - Analyzed sales by contract type

  Pivot Tables:
  - Sales by Region and Product
  - Monthly Sales by Region
  - Average Order Value by Product and Region

  Visualizations (5 charts):
  - Monthly sales trend bar chart
  - Top customers horizontal bar chart
  - Product sales pie chart
  - Regional sales comparison
  - Product sales heatmap by region
